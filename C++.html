C++

Sejarah C++:
- 1979-1983: C++ dikembangkan oleh Bjarne Stroustrup di Bell Labs sebagai perluasan dari bahasa pemrograman C. Nama "C++" pertama kali digunakan pada tahun 1983.
- 1985: C++ pertama kali dirilis secara resmi sebagai bahasa pemrograman komersial.
- 1989: Standar pertama C++ diterapkan, yang sering disebut sebagai C++ '89 atau C++ 2.0.
- 1998: Standar C++98 diperkenalkan, membawa banyak perubahan dan perbaikan signifikan.
- 2003: Standar C++03 dirilis sebagai revisi kecil dari C++98.
- 2011: Standar C++11 diperkenalkan dengan peningkatan signifikan termasuk lambda expressions, auto keyword, dan smart pointers.
- 2014: Standar C++14 dirilis sebagai perbaikan dari C++11 tanpa perubahan besar dalam fitur.
- 2017: Standar C++17 diperkenalkan dengan penambahan fitur seperti structured bindings dan filesystem library.
- 2020: Standar C++20 diperkenalkan dengan banyak fitur baru seperti Concepts, Ranges, dan peningkatan pada bahasa dan perpustakaan.
Keunggulan C++:
1. Kinerja Tinggi: C++ memberikan kendali langsung terhadap sumber daya komputer, sehingga dapat menghasilkan kode yang sangat efisien.
2. Pemrograman Berorientasi Objek: C++ mendukung pemrograman berorientasi objek, memungkinkan pengembangan dengan konsep pewarisan, polimorfisme, dan enkapsulasi.
3. Fleksibilitas dan Kontrol: C++ memberikan tingkat kontrol tinggi terhadap hardware dan memori, memberikan fleksibilitas kepada pengembang.
4. Portabilitas: Meskipun tidak setinggi Java, C++ masih dapat berjalan di berbagai platform dengan perubahan minimal.
5. Dukungan untuk Pemrograman Generik: C++ memiliki dukungan kuat untuk pemrograman generik melalui fitur-fitur seperti template.
6. Komunitas yang Kuat: C++ memiliki komunitas yang besar dan aktif, dengan banyak sumber daya dan bantuan.
Properti C++:
1. Bahasa Pemrograman Berorientasi Objek dan Prosedural: C++ adalah bahasa yang mendukung paradigma pemrograman berorientasi objek dan prosedural.
2. Memory Management: C++ memberikan kontrol langsung terhadap alokasi dan dealokasi memori, termasuk fitur manual seperti operator new dan delete.
3. Pointer dan Referensi: C++ mendukung penggunaan pointer dan referensi, memungkinkan manipulasi langsung pada alamat memori.
4. Sintaksis Mirip dengan C: Karena turunan dari C, sintaksis C++ mirip dengan C, membuatnya lebih mudah bagi pengembang yang sudah terbiasa dengan C.
5. Template: C++ memiliki sistem template yang kuat untuk pemrograman generik, memungkinkan penulisan kode yang dapat digunakan ulang dengan tipe data yang berbeda.
C++ digunakan secara luas dalam pengembangan perangkat lunak tingkat tinggi seperti game, sistem operasi, mesin pencari, dan perangkat lunak embedded yang membutuhkan kontrol langsung terhadap sumber daya.
